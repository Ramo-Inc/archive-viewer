# 要件

eagle viewer の様なIFのマンがビュワーをWindowsネイティブのRUSTでつくってほしい

## やりたい事
Eagle Viewer のようなユーザーインターフェースを用意してそこに対してジップで固まった画像ファイルのマンガを読めるようにしたい。

Eagle Viewer のようなユーザーインターフェースで表示されるのはジップで固まった画像ファイル群の先頭の画像の表紙だけを見せる。
クリックするとそのアーカイブの画像ビュアが表示されて2枚を1画面つまり見開きページをちゃんと表示できるようにしてビュアとして次ページ前ページで表示される。

基本的にはドラッグ&ドロップでアーカイブを落とせば内部で自動的に管理してくれるような仕組みにしてほしい。ここら辺はイーグルビアーと完璧に同じにしてください

ソートやお気に入り、タグや、スマートディレクトリ機能などはEagle Viewer を踏襲してほしい。

まずはオンラインリサーチで　Eagle Viewer 調査してからどのような機能があるのかよく考えて基本機能で重要なものをちゃんと吸収して使えるようにしてほしい　細かい込み入った機能は必要ないけど基本的な機能をまずインプリしてちゃんとサクサクビュアーとして動くような形でまとめてほしい
